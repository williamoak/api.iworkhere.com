# Contributor Covenant Code of Conduct

We as contributors and maintainers pledge to make participation in this project a harassment-free experience for everyone, regardless of background or identity. We expect that contributors will:

- Be respectful and civil in communications
- Assume good faith and give others the benefit of the doubt
- Use welcoming and inclusive language

If you experience or witness unacceptable behavior, please report it to the maintainers by opening an issue or contacting the repository administrators.

For the full Code of Conduct, see: https://www.contributor-covenant.org/version/2/0/code_of_conduct/
