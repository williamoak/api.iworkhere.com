# Security

If you discover a security vulnerability in this project, please report it privately to the maintainers so we can investigate and fix it before public disclosure.

Recommended reporting steps:

1. Email security@iworkhere.com (or open a private issue on GitHub if you have an organization plan) with:
   - A clear description of the vulnerability
   - Steps to reproduce (minimum reproducible example)
   - Any potential impact and suggested remediation
2. Do not post public details until a fix or mitigation is available.

We will respond to security reports within 48 hours.

Useful notes for maintainers:
- Rotate tokens and secrets if they are exposed.
- Use GitHub Secrets in CI for database credentials and API keys.
- Consider using Dependabot or Renovate for automated dependency updates.
